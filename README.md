# Apartment Management Frontend

React + TypeScript + Vite frontend for the Apartment Management system.

## Prerequisites

- Node.js 18+ (LTS recommended)
- npm

## Environment Setup

1. Copy `.env.example` to `.env`.
2. Set the environment values:
   - `VITE_API_PROXY_TARGET` (e.g., `http://localhost:8080`)

## Install Dependencies

```bash
npm install
```

## Run in Development

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
